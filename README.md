# Introduction

This project is a WooCommerce plugin to calculate the shipping costs using the eShipping API and create an order in the eShipping system.

# Getting Started

1. Installation process

- Download the .zip file from the repository.
- Upload it with the plugin uploader in the WordPress admin panel.
- Activate the plugin.
- Go to the settings page and enter the API keys.
- Fill all the settings of the plugin.
- Fill the products datas for each product.

2. Software dependencies

- WordPress 5.0 or higher
- PHP 7.0 or higher
- WooCommerce 3.0 or higher

3. Latest releases

- Fixed a bug when "ship to a different address" is checked.
- Added a filter when the product is virtual.
